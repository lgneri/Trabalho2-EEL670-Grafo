Para utilizar o programa va, atraves do terminal, ate o diretorio com os arquivos
de codigo fonte Vertice.h, Vertice.cpp, Aresta.h, Aresta.cpp, Grafo.h, Grafo.cpp,
mainGrafo.cpp e os arquivos makefile e brasil.csv. Em seguida digite no terminal
"make" para gerar o executavel do programa. Entao, digite "./grafo brasil.csv" para
executar e selecione uma das opcoes oferecidas pelo menu do programa. Ademais,
pode-se utilizar o comando "make clean" no terminal para deletar o executavel.
